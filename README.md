# Digital Sebha

A simple and responsive digital Sebha (Tasbeeh Counter) application built with Next.js. The application helps users count and track their daily dhikr through a clean and intuitive interface.

## Features

- Digital Tasbeeh Counter
- Reset Counter
- Responsive Design
- Fast Client-Side Navigation
- Modern User Interface

## Technologies

- Next.js
- React
- JavaScript
- CSS

## Purpose

This project was developed to practice building modern web applications using Next.js, React, and component-based development.

## Project Structure

```
digital-sebha-nextjs/
│
├── public/
├── src/
│   └── app/
├── package.json
├── next.config.mjs
└── README.md
```

## Getting Started

```bash
npm install
npm run dev
```

Open **http://localhost:3000** in your browser.
